# Site: Node Astro

Minimal static site using Astro, ready for deployment on Releasea.

## Local Development

```bash
npm install
npm run dev
```

The dev server starts on `http://localhost:4321`.

## Build

```bash
npm run build
```

Output is generated in the `dist/` directory.

## Preview

```bash
npm run preview
```

## Project Structure

```
site-node-astro/
  src/pages/
  astro.config.mjs
  package.json
```
