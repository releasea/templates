# Site: Node Vite

Minimal static site using Vite, ready for deployment on Releasea.

## Local Development

```bash
npm install
npm run dev
```

The dev server starts on `http://localhost:5173`.

## Build

```bash
npm run build
```

Output is generated in the `dist/` directory.

## Preview

```bash
npm run preview
```

## Project Structure

```
site-node-vite/
  src/
  index.html
  vite.config.js
  package.json
```
