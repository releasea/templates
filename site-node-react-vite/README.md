# Site: Node React Vite

Minimal React static site using Vite, ready for deployment on Releasea.

## Local Development

```bash
npm install
npm run dev
```

The dev server starts on `http://localhost:5173`.

## Build

```bash
npm run build
```

Output is generated in the `dist/` directory.

## Project Structure

```
site-node-react-vite/
  src/
  index.html
  vite.config.js
  package.json
```
