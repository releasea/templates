# Site: Node Next.js (Static Export)

Minimal static site using Next.js with static export, ready for deployment on Releasea.

## Local Development

```bash
npm install
npm run dev
```

The dev server starts on `http://localhost:3000`.

## Build

```bash
npm run build
```

Output is generated in the `out/` directory.

## Preview

```bash
npm run preview
```

## Project Structure

```
site-node-next/
  src/pages/
  next.config.mjs
  package.json
```
