# Site: Node Eleventy

Minimal static site using 11ty, ready for deployment on Releasea.

## Local Development

```bash
npm install
npm run dev
```

The dev server starts on `http://localhost:8080`.

## Build

```bash
npm run build
```

Output is generated in the `_site/` directory.

## Preview

```bash
npm run preview
```

```
site-node-eleventy/
  src/
  .eleventy.js

```
